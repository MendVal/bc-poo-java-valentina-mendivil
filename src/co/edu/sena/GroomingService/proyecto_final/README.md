## Descripción del Sistema
Mascotas Elegantes es un sistema completo de gestión para un servicio de grooming y cuidado de mascotas. El sistema permite administrar el registro de mascotas, propietarios, citas de grooming, y mantener un historial completo de todas las atenciones realizadas. El sistema gestiona diferentes tipos de mascotas y sus servicios especializados de peluquería y estética.
El sistema está diseñado siguiendo los principios de Programación Orientada a Objetos, implementando encapsulación para proteger los datos sensibles, herencia para organizar diferentes tipos de mascotas, polimorfismo para permitir comportamientos especializados, y un robusto manejo de excepciones personalizadas para garantizar la integridad de los datos. La aplicación utiliza colecciones profesionales (HashMap y ArrayList) para búsquedas eficientes y gestión ordenada de la información.
